:thinking:注意：

  1. 叹号!放在可选类型后面表示取值

  2. i++和i–在swift用不了

  3. 区间运算符

  4. === 和 *!==*在swift中用来比较两个引用是否相等。

  5. 元组的比较只比较对应位置上的是否类型和内容相同

  6. 空值运算符在swift中用法相当于简化的三目运算符

if语句

swift中判断的条件可以不用括号括起来，但推荐加括号

可选类型绑定

:thinking:注意：用变量接收a也可以，但是会给警告:warning:

  • 隐式展开：注意要给接收的变量明确指定相同的类型

switch语句

  1. swift中switch语句的case在没有处理操作时需要加break，否则可以不加break。

  2. fallthrough会使代码强制穿透到下一个条件的操作

  3. case可以匹配多种情况

  4. 区间运算符：a..<b（不包括b）a…b（包括b）

  5. 元组拆分：

    1. 可以用var或let来接收和匹配元组的值；
    2. where可以在前面var或let的基础上进一步进行条件判断

for…in

stride函数：

  1. 不包括结尾

    • 参数：from，to，by（步进值）

    • 返回：一个区间

  2. 包括结尾

    • 参数：from，through，by（步进值）
    • 返回：一个区间

反向输出：reversed方法

continue和break同C语言

while循环

普通的while循环同C语言

repeat…while同do…while循环

String

方法：

  1. count：返回字符串的长度

  2. String[String.startIndex]：返回字符串开始处的字符

  3. String[String.endIndex]：指向’\0’，会报错，不能返回字符串结束处的字符

  4. String.index(before:String.Index)：返回字符串指定位置前一字符的索引

  5. String.index(after:String.Index)：返回字符串指定位置后一字符的索引

  6. String.index(String.Index,offsetBy:String.IndexDistance)：返回字符串指定位置前或后第n个字符的索引

  7. String[startIndex…endIndex]：返回指定闭区间的字符段

    String[startIndex..<endIndex]：返回指定半开半闭区间的字符段

  8. String.firstIndex(of:Character)：返回第一次出现指定字符的索引（返回可选类型）

  9. String.prefix(String.Index)：返回从字符串开始处到指定长度的子串
