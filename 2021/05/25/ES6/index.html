ES6基础语法

发表于2021-05-25|更新于2021-06-04|前端JavaScript
|阅读量:

ES6

变量：let

  1. 不能重复声明

  2. 块级作用域

    ES5中有三种作用域：全局、函数、eval（在ES5中的strict模式才会出现）

    ES6引入了块级作用域：仅在定义的块中可用。（if、else、while、for）

  3. 不存在变量提升

    变量提升：在执行之前先收集变量和函数的声明。（变量默认值为undefined）

  4. 不影响作用域链

常量：const

  1. 一定要赋初值

  2. 常量名用大写（潜规则）

  3. 常量值不可以修改

  4. 块级作用域

  1. 对数组和对象的元素做修改，不算对常量的修改，不会报错

解构赋值

ES6允许按照一定模式从数组和对象中提取值，对变量进行赋值，称为解构赋值。

  1. 数组的解构

  2. 对象的解构

模板字符串：``

字符串声明方式：``（ES6）、’’、””

  1. 声明

  2. 内容中可以出现换行符

  3. 变量拼接 $()语法

对象简化属性写法

ES6允许在大括号里直接写入变量和函数，作为对象的属性和方法（逗号后注意换行）

还允许方法定义的简化：

箭头函数：=>

ES6允许=>定义函数

  1. this是静态的，始终指向函数声明时所在作用域下的值，而不是实例对象

  2. 不能作为构造函数来实例化对象

  3. 不能使用arguments变量

  4. 箭头函数的简写

    1. 省略小括号，当形参只有一个的时候

    2. 省略花括号和return，当代码体只有一个语句且返回表达式的结果

函数参数默认值

ES6允许给函数参数赋初始值，在没有另外传参的情况使用这个值

  1. 一般情况下，赋初值的参数位置要尽量靠后，避免实参的分配问题

  2. 可以与解构赋值、参数默认值结合

rest参数：…args

ES6引入rest参数，用于获取实参，取代ES5的arguments

  1. ES5获取实参

  1. rest参数

  2. rest参数必须放在所有参数最后

spread扩展运算符：…

ES6中的扩展运算符能将数组转换为逗号分隔的参数序列

  1. 数组的合并

  2. 数组的克隆

  3. 将伪数组转化为真数组

Symbol类型

ES6新的数据类型：Symbol，表示独一无二的值，类似于字符串。

特点：

  1. Symbol的值是唯一的，用来解决命名冲突

  2. Symbol不能和其他数据类型进行运算

  3. Symbol定义的对象属性不能用for…in循环遍历，但是可以使用Reflect.ownKeys来获取对象的所有键名

迭代器：[Symbol.iterator]

ES6新增for…of遍历方法，Iterator接口主要供for…of消费

原生具备Iterator接口的数据（可以用for of 遍历，含有Symbol.iterator内置方法）

  1. Array
  2. Arguments
  3. Set
  4. Map
  5. String
  6. TypedArray
  7. NodeList

实现原理：

  1. 创建一个指针对象，指向当前数据结构的起始位置
  2. 第一次调用对象的next方法，指针自动指向数据结构的第一个成员
  3. 接下来不断调用next方法，指针一直向后移动，直到指向最后一个成员
  4. 每次调用next方法会返回一个含有value和done属性的对象

生成器：* 和yield

生成器是一个特殊的函数，用来进行异步编程（node fs ajax mongodb）

Set类型

ES6新增了一种数据类型叫Set，类似于数组，但元素值是唯一，集合实现了iterator接口，可以使用扩展运算符和for…of遍历方法。

集合的属性和方法：

  1. size：返回集合的元素个数
  2. add：增加一个新元素，并返回当前集合
  3. delete：删除一个元素，返回Boolean值
  4. has：检测集合中是否包含某个元素，返回Boolean值
  5. clear：清空集合中的所有元素

使用Set

  1. 数组去重

  2. 并集

  1. 交集

  2. 差集

Map类型

ES6新增了一种数据类型Map，类似于对象，也是键值对的集合，但键的范围不限于字符串，各种类型（包括对象）都可以作为键是唯一，Map实现了iterator接口，可以使用扩展运算符和for…of遍历方法。

Map的属性和方法：

  1. size：返回Map的元素个数
  2. set：增加一个新元素，并返回当前Map
  3. delete：删除一个元素，返回Boolean值
  4. get：返回键名对象的键值
  5. has：检测Map中是否包含某个元素，返回Boolean值
  6. clear：清空Map

class类

ES6将ES5的构造函数改成class。

静态成员

getter和setter设置

getter可以用来动态计算结果，setter可以对重新赋值进行类型判断

数值拓展

  1. Number.EPSILON：2.220446049250313e-16

  2. 二进制和八进制

  3. Number.isFinite：检测一个数值是否为有限数

  4. Number.isNaN：检测一个数值是否为NaN

  5. Number.praseInt：字符串转整数

    Number.praseFloat：字符串转浮点数

  6. Number.isInteger：判断一个数是否为整数

  7. Math.trunc：将数字的小数部分抹掉

  8. Math.sign：判断一个数为整数、负数还是零

对象方法拓展

  1. Object.is：判断两个值是否完全相等

  2. Object.assign：对象的合并（覆盖同名属性）

  3. Object.setPrototypeOf，

模块化

模块化是将一个大的程序文件，拆分成许多小的文件，然后把小文件组合起来。

模块化的优化：

  1. 防止命名冲突

  • export用于规定模块的对外接口，暴露提供的功能
  • import用于导入其他模块提供的功能
